# pycolorpint

Simple utility wrapper on `print` for colorful text. 

```python
from pycolorpint import ColorPrinter as c

c.print("""
    [hello! this is red:red]
""")

c.print("""
    [hello! this is bold:bold]
""")
```

Supported color/feature

|Color|
|-----------|
|purple|
|cyan|
|darkcyan|
|blue|
|green|
|yellow|
|red|
|bold|
|underline|
